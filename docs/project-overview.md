# ProFairy — Медицинская экспертная система

## Что это
Автоматическая система интерпретации лабораторных анализов из PDF файлов. Определяет вероятные причины отклонений показателей на основе вероятностной модели и выдаёт понятное объяснение клиенту.

## Для кого
- Люди, которые сдали анализы и хотят понять результаты до визита к врачу
- Пациенты, которые хотят предварительно оценить своё здоровье
- В перспективе: врачи как вспомогательный инструмент

## Проблема
Люди получают PDF с анализами, видят цифры, но не понимают:
- Что означают отклонения?
- Насколько это серьёзно?
- К какому врачу обращаться?
- Какие показатели связаны между собой?

Наша система автоматически анализирует все показатели, находит паттерны (несколько показателей указывают на одну причину) и объясняет результат простым языком.

## Основные функции

### MVP:
- Загрузка PDF с анализами (только оригинальные, не сканы)
- Извлечение данных из PDF (имя, пол, показатели ОАК)
- Нормализация через БД (разные названия → единый код)
- Сравнение с медицинскими нормами (с учётом пола)
- Определение вероятных причин отклонений (вероятностная модель)
- Вывод результата в понятном виде

### Будущее:
- Все виды анализов (биохимия, моча, гормоны и т.д.)
- Учёт возраста и состояний (беременность, диабет)
- OCR для сканированных PDF
- Telegram бот
- История анализов и тренды
- AI-интерпретация результатов

## Страницы / экраны

### MVP:
1. **Главная (Tilda)** — описание сервиса + кнопка "Загрузить анализ"
2. **Результат** — список причин с описаниями и показателями

### Будущее:
- Личный кабинет (история анализов)
- Форма с дополнительной информацией (возраст, состояния)
- Дашборд с графиками трендов
- Страница сравнения анализов во времени

## Технические заметки

### Платформа:
- Backend: FastAPI (Python 3.11)
- Database: PostgreSQL 15
- Infrastructure: Docker, DigitalOcean
- CI/CD: GitHub Actions (auto-deploy on push to main)
- Frontend: Tilda (отдельная команда)
- Orchestration: N8N (планируется)

### Ограничения:
- Только оригинальные PDF (не сканы, не конвертированные картинки)
- Только русскоязычные анализы
- MVP: только общий анализ крови
- Система показывает ВЕРОЯТНЫЕ причины, не ставит диагнозы

### Интеграции:
- Tilda → наш API (загрузка PDF)
- N8N → оркестрация pipeline (планируется)
- AI-агент → оформление вывода (планируется)
